Para iniciar o programa abra a pasta paciente-api e execute os seguintes comandos no prompt:
"npm install" (caso seja a primeira vez que esteja rodando a api);
npm run dev (para rodar o servidor);

Após o servidor estar rodando, abra a pasta front e execute os seguintes comandos:
"npm install" (caso seja a primeira vez que esteja rodando o front);
"ng serve -o";
